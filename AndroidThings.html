Blinking led app

Overview

The main idea for the project Ive tried to understand how the pins of the raspberry pi board worked and try to play a little with them, from obvious reasons I was not having access to other sensors, only to a resitor, few leds and a raspberry pi board. So basically Ive implemented 3 options in the main application the first one starts blinking the led for about 10 seconds the second one turns on the led and the third one it will turn the led off.

Schematics Plan

Pre-requisites:

  • Android Studio 2.2+
  • raspberry pi 3 board(with all the necesary cables)
  • pc screen with hdmi port
  • hdmi cable
  • 1xled(blue or red)
  • 1xrezistor
  • 2xconnection cables
  • 1xbreadboar

Setup and Build plan:

  • To setup, follow these steps below.

  • Install and make sure the Android Studio is set up correctly and running
  • Make sure your raspberry pi board has Android Things installed and running you should see an ip in the left corn
  • Set up the abt to run from your cmd you need to add the path in enviroment variables
  • Connect with abt from your terminal with abt connect (the ip from the raspberry pi board)
  • After that you should be connected to your board and to be able to run your program form the Android Studio on your raspberry

    Running :

  • To be able to run you should have the fizical schema as the one from the above
  • With the given code you should be able to reproduce the app
